# 产品说明

## 这是什么
这是一个面向内部研究和选型的 AI 工具搜索工具。

它当前做两件事：
- 发现相关 AI 工具
- 发现这些工具的官方群入口

## 现在的核心变化
旧口径是：
- 给定一个具体产品
- 尽量定位唯一官网
- 从官网或官方 GitHub 找官方群

新口径是：
- 输入关键词
- 召回多个相关 AI 工具
- 再检查每个工具自己的官方群入口

## 为什么要改
像 `ChatGPT`、`agent`、`RAG` 这类关键词，本来就不适合只返回一个“唯一官方产品”。

用户真正想看的是：
- 有哪些相关工具
- 哪些工具有官方群
- 哪些工具暂时没有官方群

## 现在的结果长什么样
每个结果是一张工具卡片。

卡片可能有两种状态：
- `已发现官方群`
- `未发现官方群`

无群工具也不会消失，仍会展示：
- 名称
- 简介
- GitHub stars
- 创建时间
- 最近验证时间
- 官网 / GitHub 链接

## 什么算官方源
- 品牌产品：官网 / 官方 GitHub
- 开源项目型工具：repo README / 项目主页 / 项目官网

## 什么不收
- 第三方帖子中的群
- awesome 导航页里的外链群
- Discord / Telegram / 钉钉

## 当前限制
- 只支持微信 / QQ / 飞书
- 单次搜索默认最多 10 个工具卡片
- 没有接入付费搜索 API
