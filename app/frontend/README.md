# AI群聊发现器 Frontend

该目录是 AI群聊发现器的前端工作区，技术栈为 `React + TypeScript + Vite`。

## 本地运行

```bash
cd app/frontend
npm install
npm run dev
```

默认开发地址为 `http://127.0.0.1:5173`，通过 Vite 代理访问本地后端 `http://127.0.0.1:8000`。
